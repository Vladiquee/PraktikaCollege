АРМ «Облік проходження практик студентами ППЛ»
 Опис проєкту
Даний програмний комплекс-це автоматизоване робоче місце (АРМ), розроблене для цифровізації документообігу виробничої практики у ДНЗ «Полтавський політехнічний ліцей». Система дозволяє відмовитися від ручного заповнення паперових журналів, забезпечуючи швидкий облік та миттєву генерацію звітів.

 Основні можливості
1.Цифрова база даних: Ведення обліку студентів групи 44 та підприємств-партнерів у базі SQLite.
2.Автоматична генерація звітності: Формування готових документів Microsoft Word на основі шаблонів .docx одним натисканням.
3.Сучасний UI: Інтерфейс виконаний у темній темі з використанням бібліотеки CustomTkinter.
4.Авторизація: Захищений доступ до системи для запобігання несанкціонованому редагуванню даних.

 Технологічний стек
1.Мова програмування: Python 3.10.7.
2.Графічний інтерфейс: CustomTkinter.
3.База даних: SQLite3.
4.Робота з документами: python-docx.
5.Збірка проєкту: PyInstaller.

 Як запустити програму
Програма розповсюджується у вигляді портативного релізу. Щоб уникнути помилок доступу до бази даних (OperationalError), дотримуйтесь інструкції:
1.Завантажте архів PraktikaKarpiak.zip з розділу Releases.
2.Розпакуйте всі файли в одну папку.
3.Переконайтеся, що файли PraktikaKarpiak.exe, internship_ppl.db та template.docx знаходяться в одній директорії.
4.Запустіть файл PraktikaKarpiak.exe.

 Результати аналізу
Згідно з проведеним дослідженням у межах практики, розроблений АРМ отримав середню оцінку 3.83, значно випередивши традиційні методи обліку за швидкістю та надійністю.
